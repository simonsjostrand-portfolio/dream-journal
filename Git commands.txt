Add, commit and push:

git add .  // Add all changes in the current directory to the staging area.
git commit -m "Commit message"  // Commit changes in the staging area with a message.
git push // Push local commits to the remote branch.

Here are the VS Code terminal commands with brief explanations:

git clone <repository-url> # Clone a repository from a remote URL.

git status # Check the status of your working directory and staging area.

git pull origin <branch-name> # Pull the latest changes from the remote branch.

git checkout -b <new-branch-name> # Create and switch to a new branch.

git checkout <branch-name> # Switch to an existing branch.

git merge <branch-name> # Merge the specified branch into your current branch.

git stash # Temporarily save changes not ready for commit.

git stash apply # Reapply the most recent stashed changes.
